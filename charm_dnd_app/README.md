# Charm DnD App

Кроссплатформенное Flutter-приложение для Android и Web по описанию интерфейса Charm.

## Функции
- хранение нескольких листов персонажей локально через `shared_preferences`;
- создание и удаление персонажей;
- редактирование имени, уровня и характеристик;
- добавление новых характеристик и редактирование их названий;
- экран валюты с кнопками `+` и `-`, добавлением и переименованием валют;
- рюкзак: предметы, типы, заметки и поиск;
- способности: название, тип и описание;
- кубики d4, d6, d8, d10, d12, d20 с рандомным броском по кнопке.

## Запуск
Установи Flutter SDK, затем в папке проекта:

```bash
flutter pub get
flutter run -d chrome
```

Для Android:

```bash
flutter pub get
flutter run -d android
```

Сборка APK:

```bash
flutter build apk --release
```

Сборка Web:

```bash
flutter build web --release
```

## Важно
Figma-ссылка может требовать авторизацию/доступ. Этот проект собран по описанным функциям и в тёмной fantasy/DnD стилистике. Если экспортировать из Figma цвета, иконки и размеры, их можно быстро заменить в `lib/main.dart`.
